# Dawn Jobs

A simple job board built with Next.js and TypeScript.

## Features
- Job listings rendered from mock data
- Reusable UI components
- Search functionality
- Clean, responsive layout

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Notes
This project focuses on practical UI structure and component reuse, simulating how frontend features are built in a real production environment using existing components and patterns.
